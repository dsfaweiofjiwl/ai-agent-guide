# AI Agent 开发指南

面向开发者的 AI Agent 开发教程站点，覆盖从 LLM 基础到 Agent 架构的系统化知识，持续更新中。

## 内容

- Prompt Engineering（系统提示词设计、对比实验）
- Function Calling & Tool Use
- Agent 架构设计（多阶段编排、HITL、工具集成）
- 记忆系统（短期/长期记忆、RAG）
- AI 编码工具实战（Claude Code 配置、Skills 开发）
- 部署运维（云服务器、域名、HTTPS）
- Remotion 自动化视频生成入门

## 开发

```bash
npm install
npm run dev        # 本地开发
npm run build      # 生产构建
npm start          # 生产运行
```

## Docker 部署

```bash
docker compose up -d
```

## 技术栈

Next.js / TypeScript / SQLite / Docker / CSS（Georgia 衬线 + Inter 正文）

暖系编辑风：奶白底 #faf9f5 + 珊瑚橙 #cc785c 点缀，Georgia 衬线标题，Inter 正文，JetBrains Mono 代码块。整体风格克制温暖，像一本精排版的编程杂志。
